# SIT Notes Hub Full Website

This ZIP contains a complete static website for SIT Notes Hub.

## Files
- index.html
- privacy-policy.html

## What is included
- Professional home page
- Responsive navbar
- Student features section
- Admin dashboard features section
- Core design and security section
- How it works section
- Privacy Policy page connected in navbar

## How to publish on GitHub Pages
1. Create a GitHub repository named `sit-notes-hub-website`.
2. Upload both files:
   - index.html
   - privacy-policy.html
3. Open repository Settings.
4. Go to Pages.
5. Select "Deploy from branch".
6. Choose branch `main` and root folder.
7. Save.
8. Use the generated GitHub Pages link.
